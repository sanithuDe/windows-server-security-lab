
#  Active Directory Security Lab

This project is a **university lab** completed for the **NIT3122 – Network Design and Implementation** module at Victoria University.

##  Overview
Designed and implemented a virtual enterprise network using **Windows Server 2016** and **Active Directory**. The project focuses on user management, domain configuration, and security policy enforcement.

##  Key Features
- Configured Active Directory Domain Services (AD DS)
- Set up Domain Controllers (including RODC)
- Managed users, Organizational Units (OUs), and access control
- Implemented Group Policy for security restrictions
- Applied account lockout and authentication policies

##  Technologies
Windows Server 2016 • Active Directory • PowerShell • Group Policy • Hyper-V

##  Outcome
Gained hands-on experience in **identity management, network administration, and Active Directory security** in a simulated enterprise environment.
